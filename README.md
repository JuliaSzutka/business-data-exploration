# Business Data Analysis & Clustering

## 📌 About

This repository contains two projects focused on business data analysis and customer segmentation.

## 📂 Projects

### 📊 Exploratory Data Analysis (EDA)

* Data cleaning and preprocessing
* Data visualization and pattern analysis
* Identification of trends and relationships in business data

### 👥 Customer Segmentation (Clustering)

* Unsupervised learning techniques (clustering)
* Methods used: K-Medoids, Hierarchical Clustering (Ward), Self-Organizing Maps (SOM)
* Evaluation using Silhouette Score, Calinski-Harabasz, and Davies-Bouldin Index
* Identification and interpretation of customer segments

Detailed analysis and results are included in the project report (PDF).

## 🛠 Tools

* Python / R
* pandas, scikit-learn
* visualization libraries

## 📊 Dataset

The datasets used in this project come from built-in datasets available in R packages (e.g., `fpp3` / `tsibbledata`) and additional data provided by the course instructor.

## 👥 Team Project

Developed as part of a group academic project at AGH University of Science and Technology.
